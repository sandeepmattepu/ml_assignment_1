This project compilation and running is dependent on Apache-Ant. If it is not present in your OS please follow instructions from
http://ant.apache.org/manual/install.html

1) To enter the threshold, file location, learning rate values please open the build.xml file and at line 4,5,6 you can enter the appropriate values inside the double quotes.
2) Save build.xml file
3) Open command line and navigate to this project directory. You should be able to see "build.xml" when you type "ls" command.
3) Type "ant run" and hit enter to build and run the project.
